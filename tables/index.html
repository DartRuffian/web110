Web 110 Glossary

Term Acronym / Initialism Definition
Internet Global system of connected computer networks.
Hypertext Transfer Protocol HTTP Part of the Internet protocol suite; defines commands and services used for transmitting webpage data.
Hypertext Markup Language HTML The standard markup language for documents designed to be displayed in a web browser.
World Wide Web WWW A subset of the Internet that consists of pages that can be accessed using a Web browser.
World Wide Web Consortium W3C Internation community that develop standards for the World Wide Web.
Americans with Disabilities Act ADA Standards that require companies to offer and maintain sites that people with disabilities can use and access.
World Intellectual Property Organization WIPO Specialized organization of the United Nations and provides services to register and protect intellectual property.
Client A process that iniates contact with a server to exchange data.
Server A computer or system that provides resources, data, services, etc. to clients over a network.
Client / Server A relationship between a client or clients requesting a service and a server providing that function or service.
Multipurpose Internet Mail Extensions MIME An Internet standard that extends email messages to support non-ASCII character sets, as well as audio, video, images, and applications.
Simple Mail Transfer Protocol SMTP An Internet standard communication protocol for electronic mail.
Post Office Protocol POP3 Internet standard protocol that is used by e-mail clients to retrieve e-mail from a server.
Internet Message Access Protocol IMAP Internet standard protocol that is used by e-mail clients to retrieve e-mail from a server over a TCP/IP connection.
File Transfer Protocol FTP Standard communication protocol used to transfer computer files from a server to a client over a computer network.
Transmission Control Protocol TCP One of main protocols of the Internet protocol suite. Provides reliable, ordered, and error-checked delivery of a stream of bytes between applications on an IP network. Major internet applications run on TCP.
Internet Protocol IP The network layer communications protocol in the Internet protocol suite for relaying data across networks.
Domain Name System DNS Decentralized naming system used to identify computers reachable through the Internet or other Internet Protocol networks.
Uniform Resource Locator URL Also called a web address. A reference to a web resource that specifies its location on a computer network and a mechanism for retrieving it.
Domain Name A string that identifies a network domain, or represents an Internet Protocol resource, a server hosting a website or a website itself, or any other service communicated via the Internet.
Fully Qualified Domain Name FQDN Also called an Absolute Domain Name. A domain name that specifies its exact location in the hierarchy of the Domain Name System.
Top Level Domain TLD One of the domains at the highest level in the Domain Name System of the Internet after the root domain.
Markup Language Data included in an electronic document which is distinct from the document's content. It is typically not included in representations of the document for end users.
Element A type of HTML document component, one of several types of nodes in HTML. Elements add semantics and formatting to parts of a document. Elements are sometimes wrongly referred to as tags.
Tag Tags are used delimit the start and end of elements in the markup.
Attribute A property of an object, element, or file.
Extensible HyperText Markup Language XHTML Part of the XML markup languages. It mirrors or extends versions of HTML.
Hypertext Markup Language 5 HTML5 The fifth and final major HTML version that is recommended by W3C. The current specification is known as the HTML Living Standard
Doctument Type Definition DTD Set of markup declarations that define a document type for a SGML-family markup language. Defines the valid building blocks of am XML document.
DOCTYPE An instruction that associates a particular XML or SGML document with a document type definition.
